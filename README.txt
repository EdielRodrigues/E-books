PERSONAL TRAINER AVANÇADO PRO — PWA

COMO PUBLICAR NO GITHUB PAGES
1. Extraia o arquivo ZIP.
2. Envie todos os arquivos e a pasta icons para a raiz do repositório.
3. Em Settings > Pages, escolha Deploy from a branch e a pasta /(root).
4. Abra o endereço HTTPS gerado.
5. No celular, use o botão Instalar ou “Adicionar à tela inicial”.

IMPORTANTE
- Não mude os nomes index.html, app.js, data.js, style.css, manifest.json ou service-worker.js.
- Para atualizar o cache, altere o nome CACHE dentro de service-worker.js.
- O conteúdo é educacional e não substitui avaliação clínica.
